Redesign the existing Virtual TaskMaster mobile platform to clearly separate the system into three different roles:

1. USER
2. AVATAR
3. ENTERPRISE

Each role must have its own login flow, homepage, navigation system, and features. Once a user selects a role and logs in, the entire interface should change to only show tools relevant for that role.

The platform should feel like three specialized dashboards within one ecosystem.

Maintain the existing futuristic design language:
dark theme, neon cyan accents (#00d9ff), smooth rounded glass-like UI panels, minimal modern layout similar to TikTok, Instagram, and Apple VisionOS.

------------------------------------------------

FIRST SCREEN (ROLE SELECTION BEFORE LOGIN)

Create a landing screen where people choose how they want to use the platform.

Title:
"How will you use Virtual TaskMaster?"

Three large cards:

USER
Explore live experiences, watch reels, join avatars, and book real-world activities.

AVATAR
Become a live guide or operator performing tasks using cameras or smart glasses.

ENTERPRISE
Organizations that hire avatars for professional services, training, inspections, and remote operations.

Each card should include:
icon
short description
“Continue” button

Once a role is selected, the user proceeds to a role-specific login.

------------------------------------------------

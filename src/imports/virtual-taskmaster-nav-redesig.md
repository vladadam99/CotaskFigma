Redesign and simplify the navigation system of the existing Virtual TaskMaster mobile app.

The current navigation menu contains too many items and needs to be reorganized into a cleaner and more modern structure. Do not rebuild the entire app. Instead improve the current UI by reducing clutter and grouping features logically.

The platform has three different roles that must remain completely separated:

USER
AVATAR
ENTERPRISE

Each role must have its own homepage, navigation structure, and tools. Once a role logs in, the interface should only show features relevant to that role.

------------------------------------------------

GENERAL NAVIGATION RULES

Remove the large side navigation menu and replace it with a simple bottom navigation system with a maximum of 5 main tabs.

Secondary features should be moved inside profile or settings pages instead of the main navigation.

Keep the design modern and minimal like TikTok, Instagram, and Airbnb.

------------------------------------------------

USER APP NAVIGATION

Bottom navigation tabs:

Home
Explore
Reels
Messages
Profile

Home should show:
Live avatars
Recommended experiences
Trending activities

Explore should include:
World map explorer
Category browsing
Search for avatars or experiences

Reels should show:
Infinite vertical scrolling reels feed of avatars performing activities.

Messages should show:
Chat with avatars
Session updates

Profile should include:
Settings
Wallet
Bookings
Favorites
Achievements
Notifications

------------------------------------------------

AVATAR APP NAVIGATION

Bottom navigation tabs:

Dashboard
Missions
Live
Messages
Profile

Dashboard should show:
Earnings summary
Upcoming bookings
Level progress

Missions should show:
Available tasks
Accept mission buttons

Live should show:
Start live session button
Camera setup
Device connection (phone camera, smart glasses, 360 camera)

Messages should show:
Chat with users or companies

Profile should include:
Equipment settings
Achievements
Account settings
Session history

------------------------------------------------

ENTERPRISE APP NAVIGATION

Bottom navigation tabs:

Dashboard
Tasks
Avatars
Analytics
Company

Dashboard should show:
Overview of active tasks
Upcoming sessions
Company activity

Tasks should allow:
Creating missions
Managing active tasks
Editing task details

Avatars should show:
Available avatars
Ratings
Hiring options

Analytics should show:
Performance reports
Completed missions
Costs and activity metrics

Company should include:
Team management
Billing
Account settings

------------------------------------------------

DESIGN STYLE

Maintain the existing dark futuristic theme with neon cyan accents.
Use smooth rounded components and glass-like UI panels.
Follow a minimal modern layout similar to TikTok and Instagram.

The goal is to create a simple, intuitive, and clean navigation experience where each role sees only the features relevant to them.

Expand the existing Virtual TaskMaster mobile app design by adding the following major system features and screens. The design should keep the same dark futuristic theme with neon cyan accent (#00d9ff), smooth rounded glass UI panels, and minimalist Apple/Tesla-style interface.

The platform connects users with live human avatars who stream real-world experiences using 360° cameras, smart glasses, or phone cameras.

Add the following system features:

----------------------------------------------------

1. ROLE SELECTION SCREEN

Create a screen that appears after sign up where users choose how they will use the platform.

Title:
"How will you use Virtual TaskMaster?"

Options:

USER
Explore experiences, connect with avatars, book live sessions, watch reels.

AVATAR
Perform tasks, stream live experiences, interact with users.

Allow users to optionally enable BOTH roles.

Design with two large selectable cards with icons and short descriptions.

----------------------------------------------------

2. CAMERA SETUP / DEVICE CONNECTION

Create a camera setup screen where avatars choose what device they will use to stream.

Options should include:

360 Camera
Smart Glasses
Phone Camera
External Camera

Show connection status indicators.

Allow the user to test their video feed before going live.

Include device icons and connection buttons.

Show a preview window for the camera stream.

----------------------------------------------------

3. DISCOVERY FEED (TIKTOK / INSTAGRAM STYLE)

Create a vertical scrolling discovery feed similar to TikTok or Instagram Reels.

Users scroll through short videos showing avatars performing activities.

Each reel should display:

Avatar name
Avatar level badge
Category
Location
Like button
Comment button
Follow button
Live indicator if streaming

Users can tap a reel to:

Join a live session
View avatar profile
Book a session

The interface should feel modern, immersive and social-media inspired.

----------------------------------------------------

4. AVATAR LEVEL SYSTEM

Create a ranking system for avatars.

Levels should include:

Level 1 – Beginner
Level 2 – Verified
Level 3 – Specialist
Level 4 – Professional
Level 5 – Elite Expert

Each avatar profile should display:

Level badge
Completed sessions
Experience points
Unlocked skills
Equipment badges

Design level badges similar to gaming rank systems.

Higher levels unlock more categories and higher earnings.

----------------------------------------------------

5. ENTERPRISE / COMPANY DASHBOARD

Create a dashboard for organizations and companies using the platform.

Companies can:

Create tasks
Hire avatars
Schedule sessions
Track completed missions
Manage teams

Show sections for:

Active tasks
Scheduled sessions
Completed sessions
Analytics and performance metrics

Include company profile and team management features.

This dashboard should feel more professional but keep the same design language as the rest of the app.

----------------------------------------------------

General design style:

Dark futuristic theme
Neon cyan accent color
Glass-like UI elements
Smooth rounded cards
Modern tech aesthetic
Mobile-first design for iOS and Android

The overall product should feel like a combination of TikTok, Instagram, and a remote presence platform for real-world experiences and professional services.
